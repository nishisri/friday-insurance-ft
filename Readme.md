# Match Filter

## Technologies Used:

* Language - Java
* Test framework - Junit5
* Build tool - Gradle
* UI testing - Selenium webdriver
* Browser - Chrome driver

## How to start the application:
To run the tests simply run the command  **./gradlew test**
